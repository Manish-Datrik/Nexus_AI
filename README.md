# Nexus AI — AI-Powered HR Decision Intelligence Platform 🚀
Nexus AI is a next-generation AI-powered HR Decision Intelligence Platform designed to transform traditional HR operations into intelligent, explainable, and autonomous workflows. Instead of replacing HR professionals, Nexus AI assists them in making faster, evidence-based hiring and workforce decisions through multiple AI agents, predictive analytics, and enterprise automation.

# 🚀 Getting Started

## 1. Clone Repository

```bash
git clone https://github.com/your-username/nexus-ai.git
```

## 2. Install Dependencies

```bash
npm install
```

## 3. Configure Environment Variables

```env
OPENAI_API_KEY=

DATABASE_URL=

NEXT_PUBLIC_API_URL=
```

## 4. Run Frontend

```bash
npm run dev
```

## 5. Run Backend

```bash
uvicorn main:app --reload
```
